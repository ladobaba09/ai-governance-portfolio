## Screenshot Best Practices

1. **Clean browser:** Close unnecessary tabs, hide bookmarks bar
2. **Full width:** Capture the full application view
3. **No PII:** Ensure no personal information visible
4. **Consistent size:** Use same browser window size for all
5. **Descriptive names:** Use the naming convention above
